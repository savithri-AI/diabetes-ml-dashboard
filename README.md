# 🩺 Diabetes Prediction Dashboard

A Machine Learning project using Logistic Regression to predict 
diabetes outcomes on the Pima Indian Diabetes Dataset.

## 📊 Results
- Accuracy: 76.6%
- Correct predictions: 118 / 154
- Strongest predictor: DiabetesPedigreeFunction

## 📈 Charts Included
- Model Accuracy
- Confusion Matrix
- Glucose vs BMI Scatter Plot
- Feature Importance

## 🛠️ Tools Used
- Python
- Scikit-learn
- Matplotlib
- Chart.js (HTML dashboard)

## 🚀 How to Run
pip install matplotlib numpy
python diabetes_dashboard.py
